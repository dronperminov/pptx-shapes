TextBoxes example
==================

This example demonstrates how to add various text boxes with different fonts, alignments, colors, and formatting using the ``TextBox`` shape from the `pptx-shapes` library.

.. image:: /_static/examples/text_boxes.png
   :alt: Basic example
   :align: center

It includes:
- Support for different font families (Calibri, Arial, Times New Roman, Consolas)
- Alignment options (left, center, right)
- Multiline text with rotation
- Fill and stroke styling
